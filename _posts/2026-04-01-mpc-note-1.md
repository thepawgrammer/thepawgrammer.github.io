### 2. Negligible Function

- A function $ν : \mathbf{N} → \mathbf{R}$ is negligible if:
  - It goes to 0 faster than any inverse polynomial
- In other words:
  - For any polynomial $p(n)$,  
    $ν(n) < 1 / p(n)$ for all but finitely many $n$

👉 **Intuition**
- Practically “almost 0”
- Used to describe “negligible attack probability”

### 3. Security Parameters

| Parameter | Role | Intuition |
|----------|------|----------|
| $κ$ | Computational security | Hard to break via computation (brute-force) |
| $σ$ | Statistical security | Hard to guess (probability) |

### 4. Two Types of Attacks

- **Computational attack**
  - Break encryption via heavy computation
  - Controlled by $κ$

- **Interactive attack**
  - One-shot attempt (e.g., guessing randomness)
  - Controlled by $σ$

### 5. Security Guarantee

Security is violated with probability:

$$
2^{-\sigma} + \nu(\kappa)
$$

- $2^{-\sigma}\$: probability of lucky guess  
- $\nu(\kappa)\$: computational attack success (negligible)

### 6. Random Sampling Notation

- We use $ v \in_R D $ to denote sampling from a distribution $D$

- Example:
  - $ k \in_R (0, 1)^{\kappa} $   
    → k is a uniformly random κ-bit string  

- More generally:
  - $ v \in_R A(x) \$   
    → v is the output of randomized algorithm A on input x  

### 7. Indistinguishability

Two distributions $D_1, D_2$ are indistinguishable if:

$$
Pr[A(D_1(n)) = 1] - Pr[A(D_2(n)) = 1] \le \nu(n)
$$

👉 **Meaning**
- No algorithm can tell them apart (except negligible difference)

👉 **Note**
- If we restrict $A$ to polynomial-time algorithms → **computational indistinguishability**
- If we allow all algorithms → **statistical indistinguishability**

### 8. Types of Security

- **Computational security**
  - Based on computational indistinguishability
  - Against polynomial-time adversaries

- **Statistical (information-theoretic) security**
  - Based on statistical indistinguishability
  - Against even unbounded adversaries

👉 **Definition**
- Computational security assumes adversaries are **non-uniform polynomial-time algorithms**
- Statistical (information-theoretic) security assumes **arbitrary adversaries**, even with unbounded computational power

### 9. Statistical Distance

The statistical distance between two distributions is defined as:

$$
\Delta(D_1(n), D_2(n)) = \frac{1}{2} \sum_x \left| Pr[x = D_1(n)] - Pr[x = D_2(n)] \right|
$$

👉 **Meaning**
- Measures how different two distributions are
- Represents the maximum distinguishing advantage of any adversary
